# 🩺 SwasthSarthi – Rural Healthcare Assistance App

SwasthSarthi is a cross-platform healthcare application designed to support rural health workers in collecting patient health data, generating AI-assisted diagnoses, and managing reports even in low-connectivity areas.

Built using React Native, Expo, TypeScript, Express.js, and MongoDB.

---

## 🚀 Features

- 👩‍⚕️ Health worker authentication
- 🧾 Patient registration & medical history tracking
- ❤️ Vital health data collection
  - Heart Rate
  - Blood Pressure
  - Blood Sugar
  - Temperature
- 🤖 AI-assisted diagnosis & recommendations
- 📶 Offline-first architecture using AsyncStorage
- 🔄 Automatic sync queue when internet reconnects
- 📄 Report generation & sharing
- 🏥 Doctor review workflow
- 🌐 Bilingual support (English & Hindi)
- 📱 Cross-platform support (Android, iOS, Web)
